How It Works

1. The LogiGo overlay appears in the bottom-right corner
2. Click "Play" to start execution
3. Watch as each code block highlights in real-time
4. Use the speed slider to slow down or speed up
5. Click "Pause" or "Step" to debug line-by-line

  await LogiGo.checkpoint('step1');

}
